Readme for checkFiles.py

Checks modifying date against last time checked and emails the files to receiver specified in 
config.ini

Instructions included in py file are in Dutch, but not necessary for running.

1. Modify the config.ini [EMail_Handler] section accordingly
2. In the script, specify the path variable to the folder that is to be checked by the script.
3. After running, a txt file with the time entry of the last moment the script ran is created in the
   same folder.
